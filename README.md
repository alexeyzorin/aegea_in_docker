1. Выполни `cp -n .env.template .env` и пропиши в `.env` параметры 
2. Выполни `./init.sh` 
3. Пропиши в `caddy/Caddyfile` вместо `localhost:80` свой домен  
4. Выполни `docker-compose build`  
5. Выполни `docker-compose run`
